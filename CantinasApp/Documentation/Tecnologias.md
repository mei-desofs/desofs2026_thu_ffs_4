| Camada              | Tecnologia / Ferramenta | Finalidade / Observações                                   |
| ------------------- | ----------------------- | ---------------------------------------------------------- |
| **Frontend**        | React + Vite            | Construção do web app, Vite para bundling rápido e moderno |
| **Routing**         | React Router DOM        | Navegação entre páginas                                    |
| **Styling**         | CSS                     | Estilização rápida e responsiva                            |
| **Icons**           | Lucide React            | Biblioteca de ícones                                       |
| **HTTP Requests**   | Axios                   | Comunicação com backend                                    |
| **Backend**         | Node.js + Express       | Servidor web e APIs REST                                   |
| **Database**        | MySQL                   | Armazenamento de dados relacional                          |
| **ORM** (opcional)  | Sequelize               | Facilitar operações com MySQL                              |
| **Dev Tools**       | Postman / Insomnia      | Testar APIs                                                |
| **Version Control** | Git + GitHub            | Controle de versões                                        |

Neste projeto, vamos adotar uma **arquitetura por camadas**, separando responsabilidades entre camadas como Controllers, Services e Models, garantindo código mais organizado, manutenível e escalável. Além disso, vamos seguir os princípios **SOLID** para promover um design orientado a objetos robusto e de fácil manutenção.
